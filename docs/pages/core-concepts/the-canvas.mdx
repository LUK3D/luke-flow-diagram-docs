# The Canvas

Luke Flow Diagram provides a powerful canvas for building interactive diagrams.
The canvas is the central area where nodes and connections are displayed, allowing users to visualize and interact with their data.

## Background Grid Settings

The canvas can be customized to fit your application's needs.

User `BackgroundGridSettings` to control the appearance of the background grid, including density, visibility of lines and dots, and colors.

```dart
BackgroundGridSettings(
    xDensity: 50,
    yDensity: 50,
    showDots: true,
    lineColor: const Color.fromARGB(82, 0, 0, 0),
    lineWidth: 2,
    dotColor: const Color.fromARGB(255, 0, 0, 0),
    dotRadius: 2,
    showLines: true,
)
```
If you don't provide any `BackgroundGridSettings`, the canvas will not display a background grid with the default settings.

```dart [</>]
LukeFlowCanvas(
    bacgrkoundGridSettings: BackgroundGridSettings( // [!code focus]
        xDensity: 50,   // [!code focus]
        yDensity: 50,   // [!code focus]
        showDots: true, // [!code focus]
        lineColor: const Color.fromARGB(82, 0, 0, 0),   // [!code focus]
        lineWidth: 2,   // [!code focus]
        dotColor: const Color.fromARGB(255, 0, 0, 0),   // [!code focus]
        dotRadius: 2,   // [!code focus]
        showLines: true,    // [!code focus]
    ),
    controller: controller,
    nodeBuilder: (node) {
        return Padding(
        padding: const EdgeInsets.all(4.0),
        child: Material(
            borderRadius: BorderRadius.circular(8),
            color: Colors.black,
            child: Padding(
            padding: EdgeInsetsGeometry.all(10),
            child: Row(
                children: [
                Text(
                    "This is a node",
                    style: TextStyle(color: Colors.white),
                ),
                ],
            ),
            ),
        ));
    },
)

```
:::
As you can see in the example above, the canvas is highly customizable and can be adapted to various use cases.

## Background Grid with Secondary Settings
You can also provide a secondary background grid with different settings. This can be useful for creating a more complex visual structure.

## Example
:::code-group

<div data-title="Preview" style={{width: '100%',}}>
<Demo url='/custom-canvas-2' />
</div>

```dart [</>]
LukeFlowCanvas(
    bacgrkoundGridSettings: BackgroundGridSettings(
        xDensity: 50,
        yDensity: 50,
        showDots: true,
        lineColor: const Color.fromARGB(82, 0, 0, 0),
        lineWidth: 2,
        dotColor: const Color.fromARGB(255, 0, 0, 0),
        dotRadius: 2,
        showLines: true,
    ),
    secondaryBacgrkoundGridSettings: BackgroundGridSettings( // [!code ++]

        showLines: true, // [!code ++]
    ), // [!code ++]
    controller: controller,
    nodeBuilder: (node) {
        return Padding(
        padding: const EdgeInsets.all(4.0),
        child: Material(
            borderRadius: BorderRadius.circular(8),
            color: Colors.black,
            child: Padding(
            padding: EdgeInsetsGeometry.all(10),

                children: [
                Text(
                    "This is a node",

    },
    ),
);
```
